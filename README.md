# Student Grade Calculator & Analyzer

**A simple Python tool to manage student marks, calculate grades, and generate reports.**

---

## Features
1. **Add Student Records** (Name, Roll No, 5 Subjects)
2. **Calculate Total, Average, Grade**
3. **View All Students**
4. **Find Topper**
5. **Save Data to CSV**

---

## Technologies Used
- Python 3
- `csv` module
- Basic file handling
